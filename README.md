# EchoGuard

EchoGuard is an AI-powered emergency detection system that analyzes ambient sounds
to detect critical situations such as screams, glass breaking, and distress calls.

## Project Overview
This project includes a responsive authentication interface (Sign In / Sign Up)
built using React. The UI was converted from static HTML into a component-based
React architecture with proper state management.

## Tech Stack
- React
- JavaScript
- CSS
- HTML

## Features
- Responsive Sign In / Sign Up UI
- React state-based toggle between forms
- Mobile and desktop friendly design
- Clean component structure
